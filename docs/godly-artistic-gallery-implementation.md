# Godly艺术画廊实现文档

## 概述
将原有的静态"契合之选"插画区域升级为符合Godly设计美学的多层次艺术展示。

## 实现内容

### 1. 核心组件创建
**文件：** `/components/artistic/GodlyArtisticGallery.tsx`

实现了四层架构设计：
- **背景氛围层** - 粒子动画系统 + 渐变背景
- **主体展示层** - 视差滚动插画容器
- **装饰浮动层** - SVG装饰元素（叶子、爪印等）
- **文字叙事层** - 动态标题和副标题

### 2. 样式系统扩展
**文件：** `/app/globals.css`

新增233行Godly艺术画廊专属样式：
- `.godly-artistic-gallery` - 核心容器样式
- `.ambient-background-layer` - 背景氛围层
- `.particle-system` - 粒子动画系统
- `.main-illustration-parallax` - 主体视差层
- `.decorative-floating-elements` - 装饰元素层
- `.narrative-text-layer` - 文字叙事层

### 3. 首页集成
**文件：** `/app/page.tsx`

- 导入新组件：`import GodlyArtisticGallery from '@/components/artistic/GodlyArtisticGallery';`
- 替换原有64行静态代码为单行组件调用
- 保持原有页面结构和导航流畅性

## 技术特性

### 动画系统
- **GSAP ScrollTrigger** - 滚动触发动画
- **Framer Motion** - 交互动效
- **分层时序控制** - 精确的动画编排

### 响应式设计
- **PC端 (>1024px)** - 完整艺术效果
- **平板端 (768px-1024px)** - 适度简化
- **移动端 (<768px)** - 基础展示模式

### 性能优化
- `will-change` 属性预声明
- `transform` 和 `opacity` 优先使用
- 粒子动画CSS实现减少JS开销

## 视觉效果

### 色彩方案
- **主背景：** #F9F8F6 (保持品牌一致性)
- **文字主色：** #1A1A1A
- **装饰色彩：** 绿色(#6da060)、蓝色(#5a9ec0)、橙色(#d88a5a)

### 动效参数
- **缓动函数：** power2.out (平滑自然)
- **动画时长：** 1.2s-1.5s
- **层间延迟：** 0.3s间隔
- **交互反馈：** 悬停缩放1.02倍
